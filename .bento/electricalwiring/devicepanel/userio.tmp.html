Dedicated terminal (male) (Pic 5.2):

  • Uses 4-pin male connectors in a fixed position.

  • Connected with a 3-core cable, in the order of 1, 2, 3 pins.

Connector pin number Cable color code 1 Black (1) 2 Black (2) 3 Green/Yellow 4 NC

Opposite side (female) (Pic 5.3):

  • Female connector also uses the 3-core cable with signal, positive and negative wires connected to pins 1, 2, 3 respectively.

3 Core cable colour code and connections (Pic 5.4):

  • Pin 1: Black – Signal

  • Pin 2: Black – Positive (24v)

  • Pin 3: Yellow and Green – Negative (0v)

Purpose:

  • Simplifies cable management for easier maintenance and connections.

  • Helps to organize and avoid confusions in wiring.
