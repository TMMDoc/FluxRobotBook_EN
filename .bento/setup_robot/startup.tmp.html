Reset the Fence:

The fence is to be reset when you see the message “Fence status abnormal”

  • Press shift and DIAG button (left down)

  • Press soft button RES_CHN1

  • Yes

  • Reset

Operating the robot:

Robot should be in "normal R&D" operation in T1 mode (reduced speed). If we bend with 100% then it needs to be in T2 mode

Check gripper

Check Gripper orientation in home position

Check if Vacuum is generated (DO[9])

Check if Part sensor gets signal by holding a metal to the sensor(DI[8: Stackheight_FB])
