The workflow depends much on your application. The following workflow is a suggestion for a typical cell with not a absolutely fixed setup.

The workflow is divided into the following steps:

  • Creating a first layout In this step you arrange the components (e.g. position of the pallet and position of the parts on the pallet) in a way that you get a layout without position not reachable to the robot. You might need to do special gippers for your parts, which can be done with the laser templates and the gripper tool (→ Gripper tool)

  • If needed: Manufacturing of a new gripper

  • Calibration of the Tools

  • Measuring the components with the 3 pt measurement of the Fanuc control

  • Loading frames.dg from the controller and calculate the precise position of the components (→ Frame tool)

  • Regenerating the Flux program with the precise positions of the compoentes

  • Running the program in T2 mode to see if all motions line up as expected. If there are inaccuracies you might need to adjust the frames (see best practises)

  • Running the program in Automode (safety fence closed)
