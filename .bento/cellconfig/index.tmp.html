This section describes the various options you can use to make your Flux RoboBend configuration match your BendMaster robotic bending cell.

  • Use the Track Setup options as part of the first-time setup, to configure the position and span of the robotic linear-movement track.

  • Use the Gripper Inventory to import gripper models, and to add or remove from the set of grippers that Flux uses when computing solutions.

  • Use the Bend Cell Settings to open the Bend Cell settings and make the required input to be applied inside Bend Cell.
